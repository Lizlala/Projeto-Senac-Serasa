Sistema desktop desenvolvido em **C#/.NET** como **Projeto Integrador** do programa **Transforme-se**, uma parceria entre o **Senac** e a **Serasa Experian**.

O projeto foi criado para auxiliar uma costureira autônoma no gerenciamento de clientes, encomendas e serviços, oferecendo uma solução simples e organizada para o dia a dia do seu negócio.
